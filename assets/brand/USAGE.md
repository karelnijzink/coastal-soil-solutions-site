# Brand marks — usage

- `logo-mark.svg` — primary monogram, Deep Coastal Blue. Use on Bone/white.
- `logo-mark-reversed.svg` — white, for blue/dark surfaces.
- `favicon.svg` — simplified mark (no horizon bands) on blue tile; bands drop below 32px by design.
- Clearspace: the height of the C's counter on all sides. Minimum size 24px (use favicon variant below 32px).
- The wordmark is set live in Fraunces SemiBold next to the mark (see site header) — no separate wordmark file needed for web.
- Endorsement line for Beanstalk/FPJ packaging contexts: "Soil by Coastal Soil Solutions", mark + text, mono-colour only.
- Ferment Amber (#C98A2B) never touches the identity except in the endorsement/FPJ packaging context.
